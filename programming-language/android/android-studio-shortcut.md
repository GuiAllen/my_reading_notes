
# Shortcuts for Android studio

## MacBook

  * Reveal current file in project view: `Alt-F1 ENTER`
  * Go to definition: `CMD-b`
  * Go forward/backward: `CMD-]`/`CMD-[`
  * Go to super method: `CMD-u`
  * Find usages: `Alt-F7`
  * Find file: `CMD-Shift-o`
  * Find class: `CMD-o`
  * Generate @Override: `CMD-n`
